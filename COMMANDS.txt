MAZE AGENTS — quick command reference
everything runs from the "Machine Learning" folder
right-click the folder -> open in terminal, or just cd there

  cd "C:\Users\ardac\Desktop\Machine Learning"


=== STEP 1: pick a maze ===

open config.py and change MAZE_ID near the top

  MAZE_ID = 1   ->  16x16 (small, trains fast, start here)
  MAZE_ID = 2   ->  25x25 (medium)
  MAZE_ID = 3   ->  35x35 (hard, has spikes, takes longer)

save it, then run whatever agent you want below


=== STEP 2: run an agent ===

DQN3  (tensorflow, double dqn — upgraded DQN with overoptimism fix)
  python agents/dqn3_train.py

DQN3_CUDA  (same algorithm, GPU/speed-optimised — @tf.function + batch 128 + train every 2 steps)
  python agents/DQN3_CUDA_train.py

SAC  (stable-baselines3)
  python agents/sac_train.py

  test only (needs a saved model in Results/SAC/ already):
  python agents/sac_train.py --test-only

SARSA  (tabular)
  python agents/sarsa_train.py

PPO  (pytorch, lasse's version with behaviour cloning + masking)
  python agents/ppo_train.py

each one saves results to Results/<AGENT>/ automatically


=== STEP 3: compare everything ===

run after training at least one agent

  python compare_agents.py              <- all mazes/agents found
  python compare_agents.py --maze 1     <- just maze 1
  python compare_agents.py --maze 2
  python compare_agents.py --maze 3
  python compare_agents.py --window 100 <- smoother curves (default is 50)

saves figures to Results/comparison/:
  comparison_maze1.png    6-panel dashboard
  comparison_maze2.png
  comparison_maze3.png
  cross_maze_summary.png  all agents across all mazes (only if >1 maze trained)

the 6 panels are:
  top row:   learning curve (smoothed reward) / rolling solve rate / steps per ep
  bottom:    final reward bar / final solve% bar / kpi table

=== quick tuning (config.py) ===

DQN3:
  EPISODES        1500 default, more if not converged
  EPSILON_DECAY   lower = explores longer
  LEARNING_RATE   dont go above 0.001
  GAMMA           keep high (0.99+) bc paths are long
  TAU             0.005 (soft target update, smoother than hard sync)

SAC:
  TIMESTEPS       bump up if it hasnt converged (maze2/3 need more)
  EXIT_REWARD     keep at 200, otherwise entropy dominates
  HIDDEN_SIZE     256 default

SARSA:
  EPISODES        5000, tabular needs lots of passes
  ALPHA           0.2 learning rate
  EPSILON_DECAY   0.999

PPO:
  UPDATES         160 ppo update iterations
  ROLLOUT_STEPS   512 steps per rollout
  BC_EPOCHS       300 pre-training epochs on bfs paths (runs before ppo)
  CLIP_EPSILON    0.18, controls how much policy can change per update


=== install deps (once) ===

  pip install tensorflow numpy matplotlib stable-baselines3 gymnasium torch pandas

tensorflow needs python 3.8-3.11
stable-baselines3 pulls in pytorch automatically
pandas needed for compare_agents.py
